addCard(id, heading, content, initiallyVisible)

Adds a new MenuPanelCard card to the menu panel. A card is a block inside the menu panel.
Parameters:
Name Type Description
id string The id-attribute that is placed inside the corresponding html-tag of this card. This may later be used to hide the card or change its content.
heading string The heading of this particular card inside the panel.
content string The content of this card, formatted as html.
initiallyVisible boolean Whether the card is shown from the beginning or not.
